# 产品套餐与企业功能 FAQ

## Pro 套餐定位

Pro 套餐面向成长型团队和中小企业客户，重点提供团队协作、成员权限、审计日志、自动化集成和优先支持能力。客服可以说明 Pro 套餐适合需要多人协作、统一管理权限、追踪关键操作和接入外部系统的团队。

## Pro 套餐支持能力

1. 团队工作区：支持多个成员在同一工作区内协作，成员可以共享项目、任务和业务数据。
2. 成员权限管理：支持按角色分配访问权限，常见角色包括管理员、成员、只读成员和外部协作者。
3. 审计日志：记录登录、权限变更、关键配置修改、数据导出等操作，便于安全审计和合规追踪。
4. 高级集成：支持 Webhook、API 对接、SSO、数据同步和第三方系统集成。
5. 优先支持：Pro 客户的问题会进入优先队列，适用于业务影响较高的场景。

## 不应承诺的内容

- 不要承诺具体价格、折扣、合同条款或采购周期。
- 不要承诺客户所在行业一定满足合规要求，应建议客户联系销售或合规支持确认。
- 不要编造知识库中不存在的套餐能力。

## 推荐回复结构

1. 直接回答客户提出的功能是否支持。
2. 按客户关心的点逐项说明能力。
3. 简要说明 Pro 套餐适合什么类型团队。
4. 如果客户询问采购、合同、报价或发票，应转给销售或账单支持。

## 示例

客户：Pro 套餐是否支持团队协作、权限管理、审计日志和高级集成？

建议回复：Pro 套餐支持这些能力。团队协作方面，Pro 支持多人在同一工作区协作；权限管理方面，支持按角色分配成员权限；审计日志方面，可记录关键操作用于追踪和合规；高级集成方面，支持 Webhook、API、SSO 和数据同步等能力。整体来看，Pro 更适合需要统一管理成员、追踪关键操作并接入内部系统的团队。
